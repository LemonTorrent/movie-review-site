To run program:
    1) Run "npm install" in command line
    2) Create file ".env" in top directory for The Movie DB as NEXT_PUBLIC_THEMOVIEDB_API_KEY and for The New York Times Movie API as NEXT_PUBLIC_NEWYORKTIMES_MOVIES_API_KEY.
